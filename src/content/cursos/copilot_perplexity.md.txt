# GitHub Copilot y VsCode

Aprende a desarrollar aplicaciones usando las herramientas de Microsoft más utilizadas en el entorno empresarial.

## Motivación

- Programa más rápido y limpio gracias al asistente de IA.
- Gana ventaja en entornos donde la productividad y la calidad del código son clave.
- Domina las herramientas adoptadas por los principales equipos técnicos del mundo.

## Objetivos del Curso

- Integrar GitHub Copilot con VsCode en flujos de trabajo reales.
- Aplicar sugerencias de IA para mejorar la calidad del código, refactorización y documentación.
- Evaluar las fortalezas, limitaciones y el uso responsable de Copilot en equipos de desarrollo.
- Dominar atajos, extensiones y la gestión de espacios de trabajo en VsCode.

## Contenido

### 0. Introducción al Desarrollo Moderno con IA

- 0.1. Presentación del curso: metodología y aplicaciones profesionales.
- 0.2. Evolución de las herramientas de desarrollo: del editor de texto al agente de IA.
- 0.3. Por qué VsCode y GitHub Copilot son estándar de facto.

### 1. Primeros pasos: Instalación y configuración

- 1.1. Instalando VsCode en los principales sistemas operativos.
- 1.2. Conectando y configurando GitHub Copilot.
- 1.3. Esenciales de VsCode: ajustes, temas, extensiones.

### 2. Usando VsCode como Pro

- 2.1. Atajos y flujos de trabajo para productividad real.
- 2.2. Gestión de proyectos y espacios de trabajo.
- 2.3. Personalización de VsCode para TypeScript, Java y C#.

### 3. GitHub Copilot en la práctica

- 3.1. Generación de código a partir de prompts: mejores prácticas y limitaciones.
- 3.2. Refactorización, documentación y revisiones de código con Copilot.
- 3.3. Identificación y corrección de errores generados por Copilot.

### 4. Desarrollo de Aplicaciones Reales

- 4.1. Proyecto TypeScript de extremo a extremo.
- 4.2. Integración de lógica backend con Java o C#.
- 4.3. Uso de Copilot para testing, depuración y tareas repetitivas.

### 5. Colaboración y Calidad del Código

- 5.1. Copilot en pull requests y trabajo en equipo.
- 5.2. Uso responsable: derechos de autor, ética y seguridad.
- 5.3. Cómo medir el aumento de productividad con herramientas de IA.

### 6. Más Allá: Ecosistema de Herramientas IA

- 6.1. Comparativa con otros asistentes de programación (visión general).
- 6.2. Tendencias: el futuro de la programación asistida por IA.
- 6.3. Recursos para mantenerse actualizado como desarrollador con IA.

## Características

### Prerrequisitos

- Conocimientos básicos de TypeScript (aplicable para Java y C#).
- Experiencia profesional en desarrollo.
- Interés en maximizar la productividad con IA.

### Herramientas que Utilizaremos

- **VsCode + GitHub Copilot:** entorno de programación con IA integrada.
- **TypeScript:** lenguaje versátil y universal.
- **Java:** estándar empresarial.
- **Opcionalmente C#:** para desarrolladores .NET.
